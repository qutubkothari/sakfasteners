## Optional Sizes for Responsive Design:

### Tablet (1024px breakpoint):
- Size: 1024px × 600px
- Suffix: `-tablet.jpg`

### Mobile (768px breakpoint):
- Size: 768px × 500px
- Suffix: `-mobile.jpg`

## Optimization Tips:

- Use WebP format for 30% smaller file size
- Compress at 80-85% quality
- Ensure sRGB color space
- Test on dark/light overlay to ensure text readability
