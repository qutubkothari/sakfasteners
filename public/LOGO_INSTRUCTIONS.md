# Logo Upload Instructions

Your SAK Fasteners logo needs to be saved in this directory.

## File Requirements:

**Logo File:**
- Path: `/public/logo.png`
- Format: PNG with transparent background
- Minimum Size: 200px × 200px
- Recommended Size: 512px × 512px or higher
- Name: `logo.png` (or `logo.svg` for scalable version)

## How to Add Your Logo:

1. Save your circular SAK logo as `logo.png` in the `/public/` directory
2. Use PNG format with transparent background for best quality
3. The logo will display at 48×48px in the header (automatically scaled)
4. The same logo will be used as favicon across all devices

## Logo appears in:
- Header navigation bar (48×48px)
- Browser tab favicon
- Apple device home screen
- Search engine results (when available)

The site code is ready - just add your logo file!
